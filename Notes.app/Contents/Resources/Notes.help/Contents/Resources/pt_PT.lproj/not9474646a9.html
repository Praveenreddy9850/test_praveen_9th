Ajuda de Notas

Tome notas para não se esquecer de nada, ou mantenha fotografias e ficheiros sempre à mão. Se usa contas da Internet como iCloud ou Google com as Notas, as suas notas estão disponíveis noutros computadores ou dispositivos nos quais usa essas contas.

Para começar a primeira nota é fácil: basta abrir a aplicação Notas e começar a escrever. Para escrever uma nota nova, clique no botão Compor . A aplicação Notas usa a primeira linha da sua nota como título.

Abrir a aplicação Notas

Usar a aplicação Notas com as contas

  • Adicionar uma conta para usar com a aplicação Notas: Seleccione Notas > Adicionar conta, seleccione um tipo de conta, iCloud por exemplo, e, em seguida, digite a informação necessária. Certifique-se de que assinala a opção da aplicação Notas.

    Para adicionar a aplicação Notas a uma conta que já está a usar no Mac, escolha Notas > Contas, seleccione a conta e, em seguida, seleccione a opção Notas.

  • Desactivar Notas numa conta: Escolha Notas > Contas, seleccione a conta e, depois, desassinale a opção Notas. As notas dessa conta deixarão de ser visíveis no Mac; para voltar a vê-las, active nessa conta a opção da aplicação Notas.

  • Alterar a conta predefinida: Se usar Notas com mais de uma conta, pode escolher a conta usada para notas novas. Seleccione Notas > Conta predefinida.

Escrever notas

  • Adicionar e apagar fotografias ou outros ficheiros: Arraste um ficheiro de texto ou uma imagem (em contas Exchange, não é possível adicionar ficheiros a notas). Para apagar uma imagem ou um ficheiro, seleccione-o e prima a tecla Delete.

  • Alterar o tipo de letra: Escolha Formatação > Tipo de letra e defina um tipo de letra predefinido, ou altere o tipo de letra do texto.seleccionado. No menu Formatação, pode igualmente alterar o alinhamento e a indentação do texto.

  • Fazer listas: Para abrir rapidamente uma lista com alíneas, digite * (asterisco) e prima a barra de espaço. Para criar outro tipo de listas, seleccione Formatação > Listas.

  • Encontrar ou substituir texto: Prima Comando-F. Para substituir texto, seleccione a opção Substituir, digite o texto de substituição e, em seguida, substitua todas as ocorrências do texto original, ou procure e substitua uma ocorrência específica.

Se usar Handoff, pode alternar entre o Mac e outros dispositivos ao escrever as notas. Para abrir uma nota transferida para o Mac, clique no ícone da aplicação Notas que surge na extremidade da Dock (por predefinição, na extermidade esquerda). Para abrir uma nota transferida para o dispositivo iOS, passe o dedo para cima no ícone de actividade da aplicação Notas no ecrã bloqueado do dispositivo, ou toque na aplicação Notas no ecrã multitarefa.

Ver notas

  • Ordenar notas: Escolha Visualização > Ordenar por, e ordene por data ou por título.

  • Apagar notas: Seleccione uma ou mais notas e escolha Edição > Apagar ou prima a tecla Delete. Se a opção “Todas as notas” estiver seleccionada na lista de pastas, as notas são removidas das pastas respectivas.

  • Guardar notas como documentos PDF: Seleccione uma nota e, em seguida, seleccione Ficheiro > Exportar como PDF.

  • Manter notas na secretária: Faça duplo clique numa nota na barra lateral para abri-la noutra janela e, em seguida, seleccione Janela > Flutuante no topo.

  • Encontrar notas: Use o campo de pesquisa na parte superior da barra lateral. Se tiver mais do que uma pasta, a aplicação Notas procura só na pasta actual. Se não encontrar a nota que esperava, clique em “Todas as notas” na lista de pastas.

  • Mostrar as notas numa pasta: Seleccione a pasta na lista. Para visualizar todas as notas de uma conta, clique na pasta “todas” da conta em questão (por exemplo, “iCloud - todas”). Para ver todas as notas de todas as contas, seleccione “Todas as notas”.

Utilizar pastas

  • Ocultar ou mostrar a lista de pastas: Seleccione Visualização > “Mostrar lista de pastas”, ou Visualização > Ocultar lista de pastas.

  • Criar uma pasta: Mova o ponteiro sobre o nome de uma conta da lista e clique no símbolo + quando este aparecer. Em alternativa, seleccione Ficheiro > Nova pasta (se a opção “Todas as notas” estiver seleccionada, a pasta é criada na conta predefinida; caso contrário, é criada na conta da pasta actual).

  • Mudar o nome de uma pasta: Seleccione a pasta, clique no nome e, depois, digite um nome novo. Só pode mudar o nome das pastas por si criadas.

  • Apagar uma pasta: Seleccione a pasta e, em seguida, seleccione Edição > Apagar, ou prima a tecla Delete. Só pode apagar as pastas por si criadas.

  • Mover uma pasta: Arraste a pasta para outra conta. Para torná-la uma sub-pasta, arraste-a sobre outra pasta. Só pode mover as pastas por si criadas.

  • Mover uma nota para outra pasta: Arraste a nota da barra lateral para uma pasta.

  • Copiar uma nota para outra pasta: Mantenha premida a tecla Opção enquanto arrasta a nota.

Se criar ou editar notas desligado da Internet, as notas actualizadas ficam disponíveis nas contas quando voltar a estabelecer ligação. Se não vir as suas notas, certifique-se de que “Todas as notas” está seleccionado na lista de pastas.

As notas são armazenadas em pastas, que correspondem às contas. Se não usar Notas com contas da Internet, as notas são guardadas na pasta chamada “No meu Mac”; estas notas só estão disponíveis no Mac.

Quaisquer notas criadas anteriormente no Mail aparecem automaticamente na aplicação Notas.
