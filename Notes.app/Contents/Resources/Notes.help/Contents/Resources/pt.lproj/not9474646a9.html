Ajuda Notas

Anote coisas das quais deseja lembrar-se ou mantenha fotos e arquivos à mão. Se você usa contas da Internet como iCloud no Google no Notas, suas notas são disponibilizadas em outros computadores ou dispositivos onde essas contas são usadas.

É fácil começar a escrever sua primeira nota: abra o Notas e comece a digitar. Para escrever uma nova Nota, clique no botão Criar Nota . O Notas usa a primeira linha da sua nota como título.

Abra o Notas para mim

Use o Notas com as suas contas

  • Adicionar uma conta para usar com o Notas: Escolha Notas > Adicionar Conta, selecione um tipo de conta (como iCloud) e insira as informações da sua conta Certifique-se de que a opção Notas está selecionada para a conta.

    Para adicionar Notas a uma conta já configurada no seu Mac, escolha Notas > Contas, selecione a conta e depois selecione a opção Notas.

  • Desativar as Notas em uma conta: Escolha Notas > Contas, selecione a conta e desmarque a opção Notas. As notas dessa conta não serão mais exibidas no seu Mac; para exibi-las novamente, ative as Notas na conta.

  • Alterar a conta padrão: Se você usa o Notas com mais de uma conta, é possível escolher qual conta deve ser usada para as notas novas. Escolha Notas > Conta Padrão.

Escreva notas

  • Adicionar e apagar fotos ou outros arquivos: Arraste uma imagem ou um arquivo de texto para uma nota (não é possível adicionar arquivos a notas em contas do Exchange). Para apagar uma imagem ou um arquivo, selecione-o e pressione a tecla Apagar.

  • Alterar as fontes: Escolha Formatar > Fonte e defina uma fonte padrão para todas as notas, ou altere a fonte do texto selecionado em uma nota. No menu Formatar, você também pode alterar a maneira como o texto é alinhado ou recuado.

  • Criar listas: Para iniciar rapidamente uma lista com marcadores, digite * (asterisco) e, em seguida, pressione a barra de Espaço. Para criar outros tipos de listas, escolha Formatar > Listas.

  • Buscar ou substituir texto: Pressione Comando + F. Para substituir texto, selecione a opção Substituir, digite o texto substituto e depois substitua todas as instâncias do texto original ou busque e substitua uma instância específica.

Se você usa o Handoff, será possível alternar entre o Mac e outros dispositivos enquanto você escreve notas. Para abrir uma nota passada para o Mac, clique no ícone do Notas exibido na extremidade do Dock (lado esquerdo, por padrão). Para abrir uma nota passada para o dispositivo iOS, passe o dedo para cima a partir do ícone de atividade do Notas na tela bloqueada do dispositivo ou toque em Notas na tela multitarefa.

Visualize notas

  • Ordenar notas: Escolha Visualizar > Ordenar por e ordene por data ou título.

  • Para apagar notas: Selecione uma ou mais notas e, em seguida, escolha Editar > Apagar ou pressione a tecla Apagar. Se a opção Todas as Notas estiver selecionada na lista de notas, as notas são removidas de suas respectivas pastas.

  • Salve as notas como documentos PDF: Selecione uma nota e escolha Arquivo > Exportar como PDF.

  • Manter as notas na mesa: Clique duas vezes em uma nota na barra lateral para abri-la em uma janela separada, então escolha Janela > Flutuar no Topo.

  • Buscar notas: Use o campo de busca na parte superior da barra lateral. Se você tiver mais de uma pasta, o Notas buscará apenas na pasta selecionada. Se uma nota esperada não for encontrada, clique em Todas as Notas na lista de pastas.

  • Mostras as notas em uma pasta: Selecione a pasta na lista de pastas. Para ver todas as notas de uma conta, selecione a pasta Tudo da conta, como por exemplo Tudo iCloud. Para ver todas as notas em todas as contas, selecione Todas as Notas.

Use pastas

  • Mostrar ou ocultar a lista de pastas: Escolha Visualizar > Mostrar Lista de Pastas ou Visualizar > Ocultar Lista de Pastas.

  • Para criar uma pasta: Mova o cursor sobre um nome de conta na lista e, sem seguida, clique no sinal + que aparece. Ou escolha Arquivo > Nova Pasta (se a opção Todas as Notas estiver selecionada, a pasta será criada na conta padrão. Caso contrário, será criada na conta da pasta atual).

  • Renomear uma pasta: Selecione a pasta, clique no nome e depois digite um novo nome. É possível renomear apenas as pastas que você criou.

  • Apagar uma pasta: Selecione a pasta e, em seguida, escolha Editar > Apagar ou pressione a tecla Apagar. É possível apagar apenas as pastas que você criou.

  • Mover uma pasta: Arraste a pasta para outra conta. Para transformá-la em subpasta, arraste-a sobre outra pasta. É possível mover apenas as pastas que você criou.

  • Para mover uma nota a outra pasta: Arraste a nota da barra lateral sobre uma pasta.

  • Para copiar uma nota a outra pasta: Mantenha a tecla Opção pressionada ao arrastar a nota.

Se você criar ou editar notas enquanto estiver desconectado da Internet, as notas atualizadas estarão disponíveis nas suas contas depois que você se conectar. Caso não veja suas notas, verifique se a opção Todas as Notas está selecionada na lista de pastas.

Suas notas são armazenadas em pastas que correspondem às suas contas. Se você não usa o Notas com nenhuma conta, suas notas serão armazenadas na pasta chamada Em Meu Mac; essas notas estão disponíveis apenas no seu Mac.

Quaisquer notas que você tenha criado anteriormente no Mail aparecerão automaticamente no Notas.
