Aiuto Note

Annota le cose da ricordare o tieni a portata di mano foto e file. Se utilizzi account Internet come iCloud o Google con Note, le tue note saranno disponibili sugli altri computer o dispositivi sui quali utilizzi tali account.

Iniziare con la prima nota è facile: apri Note e inizia a scrivere. Per creare una nuova nota, fai clic sul pulsante Componi . Note utilizza la prima riga della nota come titolo.

Apri Note per me

Utilizzare Note con i propri account

  • Aggiungi un account da utilizzare con Note: scegli Note > Aggiungi account, seleziona un tipo di account come iCloud, quindi inserisci le informazioni dell'account. Assicurati che il riquadro Note dell'account sia selezionato.

    Per aggiungere Note a un account già in uso sul Mac, scegli Note > Account, seleziona l'account, quindi seleziona il riquadro Note dell'account.

  • Per disattivare Note per un account: scegli Note > Account, seleziona l'account, quindi deseleziona il riquadro Note. Le note dell'account non verranno più mostrate sul Mac; perché vengano visualizzate di nuovo, attiva Note per l'account.

  • Modificare l'account di default: se utilizzi note con più di un account, puoi scegliere quale account utilizzare per le nuove note. Scegli Note > Account di default.

Scrivere le note

  • Per aggiungere o eliminare immagini o altri file: trascina un'immagine o un file di testo su una nota (non è possibile aggiungere file alle note negli account Exchange). Per eliminare un'immagine o un file, selezionali, quindi premi il tasto Cancella.

  • Per modificare i font: scegli Formato > Font, quindi imposta un font di default per tutte le note o modifica il font del testo selezionato in una nota. Dal menu Formato, puoi anche modificare l'allineamento e il rientro del testo.

  • Per creare elenchi: per creare velocemente un elenco puntato, digita * (asterisco), quindi premi la barra spaziatrice. Per creare altri tipi di elenco, scegli Formato > Elenchi.

  • Per trovare o sostituire del testo: premi Comando-F. Per sostituire del testo, seleziona il riquadro Sostituisci, inserici un testo sostitutivo, quindi scegli se sostituire tutte le occorrenze del testo originale oppure trovare e sostituire una sua occorrenza specifica.

Se utilizzi Handoff, puoi passare dal tuo Mac ad altri dispositivi e viceversa durante la creazione di note. Per aprire una nota trasferita trasferita al tuo Mac, fai clic sull'icona Note visualizzata alla fine del Dock (di default, a sinistra). Per aprire una nota trasferita al tuo dispositivo iOS, scorri verso l'alto dall'icona dell'attività di Note sullo schermo bloccato del dispositivo oppure tocca Note sullo schermo multitasking.

Visualizzare le note

  • Per ordinare le note: scegli Vista > Ordina per, quindi ordina per data o per titolo.

  • Elimina le note: seleziona una o più note, quindi scegli Modifica > Elimina oppure premi il tasto Elimina. Se l'opzione “Tutte le note” è selezionata nell'elenco cartelle, le note vengono rimosse dalle rispettive cartelle.

  • Per salvare le note come documenti PDF: seleziona una nota, quindi scegli File > Esporta come PDF.

  • Per mantenere le note sulla scrivania: fai doppio clic su una nota nella barra laterale per aprirla in una finestra separata, quindi scegli Finestra > Sempre in primo piano.

  • Per trovare le note: utilizza il campo di ricerca nella parte superiore della barra laterale. Se disponi di più di una cartella, Note effettuerà la ricerca solo nella cartella attualmente selezionata. Se non trovi una nota che ti aspettavi di trovare, fai clic su “Tutte le note” nell'elenco cartelle.

  • Per mostrare le note in una cartella: seleziona la cartella nell'elenco cartelle. Per vedere tutte le note di un account, seleziona la cartella Tutto dell'account, per esempio “Tutto iCloud”. Per vedere tutte le note di tutti gli account, seleziona “Tutte le note”.

Utilizzare le cartelle

  • Per mostrare o nascondere l'elenco cartelle: scegli Vista > Mostra elenco cartelle oppure Vista > Nascondi elenco cartelle.

  • Crea una cartella: sposta il puntatore sul nome dell'account nell'elenco, quindi fai clic sul simbolo + che appare. Oppure, scegli File > Nuova cartella (se l'opzione “Tutte le note” è selezionata, la cartella viene creata nell'account di default; altrimenti, viene creata nell'account della cartella attuale).

  • Per rinominare una cartella: seleziona la cartella, fai clic sul nome, quindi inserisci un nuovo nome. Puoi rinominare solo le cartelle create da te.

  • Per eliminare una cartella: seleziona la cartella, quindi scegli Modifica > Elimina oppure premi il tasto Elimina. Puoi eliminare solo le cartelle create da te.

  • Per spostare una cartella: trascina la cartella in un altro account. Per trasformarla in una sottocartella, trascinala su un'altra cartella. Puoi spostare solo le cartelle create da te.

  • Sposta una nota in un'altra cartella: trascina la nota dalla barra laterale su una cartella.

  • Copia una nota in un'altra cartella: tieni premuto il tasto Opzione mentre trascini la nota.

Se crei o modifichi note mentre sei sconnesso da Internet, le note aggiornate sono disponibili negli account alla connessione successiva. Se le note non sono visibili, assicurati che l'opzione “Tutte le note” sia selezionata nell'elenco cartelle.

Le note sono archiviate in cartelle, corrispondenti ai tuoi account. Se non utilizzi Note con gli account, le note vengono archiviate nella cartella “Sul mio Mac”; tali note sono disponibili solo sul Mac.

Qualsiasi nota creata precedentemente in Mail viene automaticamente visualizzata in Note.
