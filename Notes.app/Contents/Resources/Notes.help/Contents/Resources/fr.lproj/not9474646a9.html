Aide Notes

Notez les choses dont vous devez vous souvenir, ou gardez des photos et des fichiers à portée de main. Si vous utilisez des comptes Internet tels qu’iCloud ou Google avec Notes, vos notes sont disponibles sur tous les ordinateurs et appareils avec lesquels vous utilisez ces comptes.

Il est facile de créer votre première note : ouvrez Notes et commencez votre saisie. Pour rédiger une nouvelle note, cliquez sur le bouton Composer . Notes utilise la première de votre note en tant que titre.

Ouvrir Notes

Utiliser Notes avec vos comptes

  • Ajouter un compte à utiliser avec Notes : Choisissez Notes > Ajouter un compte, sélectionnez un type de compte, tel qu’iCloud, puis saisissez les informations de votre compte. Assurez-vous que la case Notes est cochée pour le compte.

    Pour activer Notes pour un compte déjà utilisé sur votre Mac, choisissez Notes > Comptes, sélectionnez le compte de votre choix, puis cochez la case Notes pour ce compte.

  • Désactiver Notes pour un compte : Choisissez Notes > Comptes, sélectionnez le compte, puis décochez la case Notes. Les notes de ce compte ne s’affichent plus sur votre Mac. Pour les afficher à nouveau, activez Notes pour le compte.

  • Modifier le compte par défaut : Si vous utilisez Notes avec plus d’un compte, vous pouvez choisir le compte à utiliser pour les nouvelles notes. Choisissez Notes > Compte par défaut.

Rédiger des notes

  • Ajouter et supprimer des photos ou d’autres fichiers : Faites glisser une image ou un fichier texte sur une note (vous ne pouvez pas ajouter de fichiers aux notes des comptes Exchange). Pour supprimer une image ou un fichier, sélectionnez l’élément à supprimer, puis appuyez sur la touche Suppr.

  • Modifier la police : Choisissez Format > Police, puis définissez une police par défaut pour toutes vos notes. Vous pouvez également modifier la police d’une portion de texte sélectionnée dans une note. Dans le menu Format, vous pouvez également modifier l’alignement et l’indentation du texte.

  • Créer des listes : Pour commencer rapidement une liste à puces, saisissez * (un astérisque) et appuyez sur la barre d’espace. Pour créer d’autres types de liste, sélectionnez Format > Listes.

  • Rechercher ou remplacer du texte : Appuyez sur Commande + F. Pour remplacer du texte, cochez la case Remplacer, saisissez le texte de remplacement, puis choisissez de remplacer toutes les instances du texte d’origine ou de rechercher puis remplacer une instance spécifique.

Si vous utilisez Handoff, vous pouvez alterner entre votre Mac et vos appareils iOS pour rédiger des notes. Pour ouvrir une note transférée vers votre Mac, cliquez sur l’icône Notes qui apparaît à la fin du Dock (par défaut, il s’agit de l’extrémité gauche). Pour ouvrir une note transférée vers votre appareil iOS, balayez vers le haut à partir de l’icône d’activité de la note sur l’écran verrouillé de l’appareil ou touchez Notes sur l’écran multitâche.

Afficher les notes

  • Trier les notes : Choisissez Présentation > Trier par, puis choisissez de trier vos notes par date ou par titre.

  • Supprimer des notes : Sélectionnez une ou plusieurs notes, puis choisissez Édition > Supprimer, ou appuyez sur la touche Suppr. Si l’option Toutes les notes est sélectionnée dans la liste de dossiers, les notes sont supprimées de leurs dossiers respectifs.

  • Enregistrer une note en tant que document PDF : Sélectionnez une note, puis choisissez Fichier > Exporter au format PDF.

  • Conserver des notes au premier plan sur le bureau : Cliquez deux fois sur une note dans la barre latérale pour l’ouvrir dans une fenêtre distincte, puis choisissez Fenêtre > Placer au premier plan.

  • Trouver des notes : Utilisez le champ de recherche situé en haut de la barre latérale. Si vous avez plusieurs dossiers, Notes effectue la recherche uniquement dans le dossier actuellement sélectionné. Si une note attendue n’a pas été trouvée, cliquez sur Toutes les notes dans la liste de dossiers.

  • Afficher les notes dans un dossier : Sélectionnez le dossier dans la liste de dossiers. Pour afficher toutes les notes d’un compte, sélectionnez le dossier Tous pour ce compte, par exemple Tous iCloud. Pour afficher toutes les notes de tous les comptes, sélectionnez Toutes les notes.

Utiliser des dossiers

  • Afficher ou masquer la liste de dossiers : Choisissez Présentation > Afficher la liste de dossiers ou Présentation > Masquer la liste de dossiers.

  • Créer un dossier : Déplacez le pointeur sur un nom de compte dans la liste, puis cliquez sur le + qui s’affiche. Sinon, choisissez Fichier > Nouveau dossier (si l’option Toutes les notes est sélectionnée, le dossier est créé dans le compte par défaut, autrement, il est créé dans le compte du dossier en cours).

  • Renommer un dossier : Sélectionnez le dossier, cliquez sur le nom, puis saisissez un nouveau nom. Vous pouvez uniquement renommer les dossiers que vous avez créés.

  • Supprimer un dossier : Sélectionnez le dossier, puis choisissez Édition > Supprimer ou appuyez sur la touche de suppression. Vous pouvez uniquement supprimer les dossiers que vous avez créés.

  • Déplacer un dossier : Faites glisser le dossier vers un autre compte. Pour en faire un sous-dossier, faites-le glisser dans un autre dossier. Vous pouvez uniquement déplacer les dossiers que vous avez créés.

  • Déplacer une note vers un autre dossier : Faites glisser la note de la barre latérale vers un dossier.

  • Copier une note vers un autre dossier : Maintenez la touche Option enfoncée tout en faisant glisser la note.

Si vous créez ou modifiez des notes alors que vous n’êtes pas connecté à Internet, les notes sont disponibles pour vos comptes lorsque vous vous reconnectez. Si vos notes n’apparaissent pas, assurez-vous que l’option Toutes les notes est sélectionnée dans la liste de dossiers.

Vos notes sont stockées dans des dossiers, correspondant à vos comptes. Si vous n’utilisez Notes avec aucun compte, vos notes sont stockées dans le dossier nommé Sur mon Mac. Ces notes sont uniquement disponibles sur votre Mac.

Si vous avez créé des notes dans Mail auparavant, celles-ci apparaissent automatiquement dans Notes.
