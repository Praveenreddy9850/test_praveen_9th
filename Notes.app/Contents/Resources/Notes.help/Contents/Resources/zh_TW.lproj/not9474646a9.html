備忘錄輔助說明

記下您要記住的事項，或讓照片和檔案隨手可用。若您搭配 Internet 帳號（例如 iCloud 或 Google）來使用「備忘錄」，則可在任何使用這些帳號的電腦或裝置上取用備忘錄。

您可以很輕易地開始使用備忘錄，只要打開「備忘錄」並開始打字即可。若要編寫新的備忘錄，請按一下「編寫」按鈕 。「備忘錄」會使用備忘錄的第一行使為標題。

為我打開「備忘錄」

搭配帳號來使用「備忘錄」

  • 加入使用「備忘錄」的帳號：選擇「備忘錄」>「加入帳號」，選擇帳號類型（例如 iCloud），然後輸入您的帳號資訊。請確定已勾選帳號的「備忘錄」註記框。

    若要為已在 Mac 上使用的帳號加入「備忘錄」，請選擇「備忘錄」>「帳號」，選擇帳號，然後勾選帳號的「備忘錄」註記框。

  • 關閉帳號的「備忘錄」：選擇「備忘錄」>「帳號」，選擇帳號，然後取消勾選「備忘錄」註記框。該帳號的備忘錄不會再顯示於您的 Mac 上；若要再次顯示備忘錄，請開啟帳號的備忘錄。

  • 更改預設帳號：若您在多個帳號中使用「備忘錄」，您可以選擇哪一個帳號來使用新的備忘錄。請選擇「備忘錄」>「預設帳號」。

編寫備忘錄

  • 加入和刪除圖片或其他檔案：將影像或文字檔案拖到備忘錄上（您無法將檔案加入 Exchange 帳號的備忘錄）。若要刪除影像或檔案，請將其選取然後按下 Delete 鍵。

  • 更改字體：選擇「格式」>「字體」，然後設定所有備忘錄的預設字體，或者更改備忘錄中已選取之文字的字體。您也可以在「格式」選單中更改文字對齊或縮排的方式。

  • 製作列表：若要快速製作帶有項目符號的列表，請輸入 *（星號），然後按下空白鍵。若要製作其他類型的列表，請選擇「格式」>「列表」。

  • 尋找或取代文字：按下 Command + F 鍵。若要取代文字，請勾選「取代」註記框，輸入取代的文字，然後取代所有的原始文字，或尋找並取代詩定的文字。

如果您使用 Handoff，您可以在編寫備忘錄時在 Mac 和其他裝置之間切換。若要打開傳送到 Mac 的備忘錄，請按一下顯示在 Dock 末端的（預設位於最左端）「備忘錄」圖像。若要打開傳送到 iOS 裝置的備忘錄，請從裝置的鎖定畫面將「備忘錄」活動的圖像往上滑，或是在多工處理畫面上點一下「備忘錄」。

檢視備忘錄

  • 排序備忘錄：選擇「顯示方式」>「排序方式」，然後依照日期或標題來排序。

  • 刪除備忘錄：選取一或多個備忘錄，然後選擇「編輯」>「刪除」，或按下 Delete 鍵。如果在檔案夾列表中選擇了 「所有備忘錄」，備忘錄會從各自的檔案夾中移除。

  • 將備忘錄儲存為 PDF 文件：選取備忘錄，然後選擇「檔案」>「輸出為 PDF」。

  • 將備忘錄保留在桌面上：在側邊欄中按兩下備忘錄，在獨立視窗中打開備忘錄，然後選擇「視窗」>「浮動於上方」。

  • 尋找備忘錄：使用側邊欄上方的搜尋欄位。若您有多個檔案夾，「備忘錄」只會搜尋目前選取的檔案夾。若找不到您要的備忘錄，請在檔案夾列表中按一下「所有備忘錄」。

  • 顯示檔案夾內的備忘錄：在檔案夾列表中選擇檔案夾。若要檢視一個帳號中的所有備忘錄，請選擇該帳號的「所有」檔案夾，例如「所有 iCloud」。若要檢視所有帳號中的所有備忘錄，請選擇「所有備忘錄」。

使用檔案夾

  • 顯示或隱藏檔案夾列表：選擇「顯示方式」>「顯示檔案夾列表」，或選擇「顯示方式」>「隱藏檔案夾列表」。

  • 製作檔案夾：將指標移到列表中的帳號名稱上方，然後按一下顯示的 +。或選擇「檔案」>「新增檔案夾」（若未選擇「所有備忘錄」，則檔案夾會製作在預設帳號中；否則，檔案夾會製作在現用檔案夾的帳號中。）

  • 重新命名檔案夾：選擇檔案夾，按一下名稱，然後輸入新的名稱。您只能重新命名由您製作的檔案夾。

  • 刪除檔案夾：選取檔案夾，然後選擇「編輯」>「刪除」，或按下 Delete 鍵。您只能刪除由您製作的檔案夾。

  • 搬移檔案夾：將檔案夾拖移至另一個帳號。若要使其成為子檔案夾，請將它拖入其他檔案夾內。您只能搬移由您製作的檔案夾。

  • 將備忘錄移至其他檔案夾：將檔案夾從側邊欄拖到檔案夾上。

  • 將備忘錄拷貝至其他檔案夾：按住 Option 鍵並拖移檔案夾。

若您在未連接 Internet 時製作或編輯備忘錄，重新連接 Internet 後，您的帳號取得更新的備忘錄。若您沒有看到備忘錄，請確認已在檔案夾列表中選擇「所有備忘錄」。

您的備忘錄會對應您的帳號儲存在檔案夾中。若您沒有在任何帳號中使用「備忘錄」，您的備忘錄會儲存在一個名為「在我的 Mac 上」的檔案夾，這些備忘錄只能在您的 Mac 上取用。

您先前在「郵件」中製作的備忘錄會自動顯示在「備忘錄」中。
