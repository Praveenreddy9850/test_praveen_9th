메모 도움말

기억해야할 일을 적어두거나 사진 및 파일을 간편하게 보관하십시오. iCloud 또는 Google과 같은 인터넷 계정을 메모와 함께 사용하는 경우 이러한 계정을 사용하는 다른 컴퓨터나 장비에서 메모를 사용할 수 있습니다.

첫 번째 메모 간단하게 시작하기: 메모를 연 다음 입력을 시작하십시오. 새로운 메모를 작성하려면 작성 버튼을 클릭하십시오. 메모는 메모의 첫 번째 줄을 제목으로 사용합니다.

메모 열기

계정으로 메모 사용하기

  • 계정을 추가하여 메모 사용하기: 메모 > 계정 추가를 선택하고 iCloud 등의 계정 유형을 선택한 다음 계정 정보를 입력하십시오. 계정에 대한 메모 체크상자가 선택되어 있는지 확인하십시오.

    Mac에서 이미 사용 중인 계정에 메모를 추가하려면 메모 > 계정을 선택하고 해당 계정을 선택한 다음 계정에 대한 메모 체크상자를 선택합니다.

  • 계정에 대한 메모 끄기: 메모 > 계정을 선택하고 계정을 선택한 다음 메모 체크상자의 선택을 해제합니다. 해당 계정에 대한 메모가 더 이상 Mac에 표시되지 않습니다. 다시 표시하려면 계정에 대한 메모를 켜십시오.

  • 기본 계정 변경: 둘 이상의 계정에 메모를 사용하는 경우 새 메모에 사용할 계정을 선택할 수 있습니다. 메모 > 기본 계정을 선택합니다.

메모 쓰기

  • 그림 또는 다른 파일 추가 및 삭제: 이미지 또는 텍스트 파일을 메모에 드래그합니다(Exchange 계정에서는 파일을 메모에 추가할 수 없음). 이미지 또는 파일을 삭제하려면 해당 이미지나 파일을 삭제한 다음 Delete 키를 누르십시오.

  • 서체 변경: 포맷 > 서체를 선택한 다음 모든 메모에 기본 서체를 설정하거나 메모에서 선택한 텍스트에 대한 서체를 변경합니다. 포맷 메뉴에서 텍스트 정렬 또는 들여쓰기 방법도 변경할 수 있습니다.

  • 목록 만들기: 구분점 표시 목록을 빠르게 시작하려면 *(별표)를 입력 후 스페이스 바를 누르십시오. 다른 유형의 목록을 생성하려면 포맷 > 목록을 선택하십시오.

  • 텍스트 찾기 또는 바꾸기: Command-F를 누르십시오. 텍스트를 대치하려면 대치 체크상자를 선택하고 대치할 텍스트를 입력한 다음 원래 텍스트의 모든 인스턴스를 바꾸거나 특정 인스턴스를 찾아 대치합니다.

Handoff를 사용하는 경우 메모를 작성할 때 Mac과 다른 장비를 전환할 수 있습니다. Mac에서 메모를 이어서 작성하려면 Dock의 끝 부분(기본적으로 맨 왼쪽에 있음)에 있는 메모 아이콘을 클릭하십시오. iOS 장비에서 메모를 이어서 작성하려면 장비의 잠금 화면에서 메모 활동 아이콘을 쓸어올리거나 멀티태스킹 화면에서 메모를 탭하십시오.

메모 보기

  • 메모 정렬하기: 보기 > 다음으로 정렬을 선택한 다음 날짜 또는 제목으로 정렬합니다.

  • 메모 삭제하기: 하나 이상의 메모를 선택한 다음 편집 > 삭제를 선택하거나 Delete 키를 누르십시오. 폴더 목록에서 모든 메모가 선택된 경우 해당 메모들이 관련 폴더에서 제거됩니다.

  • 메모를 PDF 도큐멘트로 저장하기: 메모를 선택한 다음 파일 > PDF로 보내기를 선택하십시오.

  • 데스크탑에 메모 띄우기: 사이드바에서 메모를 이중 클릭하여 별도의 윈도우에서 메모를 연 다음 윈도우 > 상단에 띄우기를 선택하십시오.

  • 메모 찾기: 사이드바 상단에 있는 검색 필드를 사용합니다. 둘 이상의 폴더가 있는 경우 메모는 현재 선택한 폴더만 검색합니다. 예상한 메모가 발견되지 않은 경우 폴더 목록에서 모든 메모를 클릭하십시오.

  • 폴더에서 메모 표시하기: 폴더 목록에서 해당 폴더를 선택합니다. 계정에 있는 모든 메모를 보려면 모든 iCloud처럼 해당 계정에 대한 모든 폴더를 선택하십시오. 모든 계정에서 모든 메모를 보려면 모든 메모를 선택하십시오.

폴더 사용하기

  • 폴더 목록 표시하기 또는 가리기: 보기 > 폴더 목록 표시 또는 보기 > 폴더 목록 가리기를 선택합니다.

  • 폴더 생성하기: 목록에서 계정 이름 위로 포인터를 이동한 다음 나타나는 +를 클릭하십시오. 또는 파일 > 새로운 폴더를 선택하십시오.(모든 메모가 선택된 경우 해당 폴더가 기본 계정 안에 생성되고 그렇지 않으면 현재 폴더의 계정 안에 생성됩니다.)

  • 폴더 이름 변경하기: 해당 폴더를 선택하고 이름을 클릭한 다음 새로운 이름을 입력하십시오. 생성한 폴더 이름만 변경할 수 있습니다.

  • 폴더 삭제하기: 폴더를 선택한 다음 편집 > 삭제를 선택하거나 Delete 키를 누르십시오. 생성한 폴더만 삭제할 수 있습니다.

  • 폴더 이동하기: 폴더를 다른 계정으로 드래그하십시오. 폴더를 하위 폴더로 만들려면 다른 폴더로 드래그하십시오. 생성한 폴더만 이동할 수 있습니다.

  • 메모를 다른 폴더로 드래그하기: 사이드바에 있는 메모를 다른 폴더로 드래그하십시오.

  • 메모를 다른 폴더로 복사하기: 메모를 드래그하는 동안 Option 키를 누르십시오.

인터넷에 연결되어 있지 않을 때 메모를 작성하거나 편집한 경우 다시 연결한 후 업데이트된 메모를 계정에서 사용할 수 있습니다. 메모가 보이지 않는 경우 폴더 목록에서 모든 메모가 선택되어 있는지 확인하십시오.

메모가 계정에 맞는 폴더에 저장됩니다. 메모에서 계정을 사용하지 않는 경우 나의 Mac이라는 폴더에 메모가 저장됩니다. 이러한 메모는 Mac에서 사용할 수 있습니다.

이전에 Mail에서 생성된 모든 메모가 자동으로 메모에 나타납니다.
