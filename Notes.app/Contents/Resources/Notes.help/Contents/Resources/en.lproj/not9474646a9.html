Notes Help

Jot down things to remember, or keep photos and files handy. If you use Internet accounts like iCloud or Google with Notes, your notes are available on other computers or devices where you use those accounts.

It’s easy to get started on your first note: open Notes and begin typing. To write a new note, click the Compose button . Notes uses the first line of your note as the title.

Open Notes for me

Use Notes with your accounts

  • Add an account to use with Notes: Choose Notes > Add Account, select an account type such as iCloud, then enter your account information. Make sure the Notes checkbox is selected for the account.

    To add Notes for an account already being used on your Mac, choose Notes >  Accounts, select the account, then select the Notes checkbox for the account.

  • Turn off Notes for an account: Choose Notes > Accounts, select the account, then deselect the Notes checkbox. The notes for that account are no longer shown on your Mac; to show them again, turn on Notes for the account.

  • Change the default account: If you use Notes with more than one account, you can choose which account to use for new notes. Choose Notes > Default Account.

Write notes

  • Add and delete pictures or other files: Drag an image or text file onto a note (you can’t add files to notes in Exchange accounts). To delete an image or file, select it, then press the Delete key.

  • Change fonts: Choose Format > Font, then set a default font for all of your notes, or change the font for selected text in a note. From the Format menu, you can also change how text is aligned or indented.

  • Make lists: To quickly start a bulleted list, type * (asterisk), then press the Space bar. To create other types of lists, choose Format > Lists.

  • Find or replace text: Press Command-F. To replace text, select the Replace checkbox, type the replacement text, then replace all instances of the original text, or find and replace a specific instance of it.

If you use Handoff, you can switch between your Mac and other devices while writing notes. To open a note handed off to your Mac, click the Notes icon that appears at the end of the Dock (by default, the left end). To open a note handed off to your iOS device, swipe up from Note’s activity icon on the device’s lock screen, or tap Notes on the multitasking screen.

View notes

  • Sort notes: Choose View > Sort By, then sort by date or title.

  • Delete notes: Select one or more notes, then choose Edit > Delete, or press the Delete key. If All Notes is selected in the folders list, the notes are removed from their respective folders.

  • Save notes as PDF documents: Select a note, then choose File > Export as PDF.

  • Keep notes on the desktop: Double-click a note in the sidebar to open it in a separate window, then choose Window > Float On Top.

  • Find notes: Use the search field at the top of the sidebar. If you have more than one folder, Notes searches only the currently selected folder. If an expected note isn’t found, click All Notes in the folders list.

  • Show the notes in a folder: Select the folder in the folders list. To see all notes in an account, select the All folder for the account, such as All iCloud. To see all notes in all accounts, select All Notes.

Use folders

  • Show or hide the folders list: Choose View > Show Folders List or View > Hide Folders List.

  • Create a folder: Move the pointer over an account name in the list, then click the + that appears. Or choose File > New Folder (if All Notes is selected, the folder is created in the default account; otherwise, it’s created in the account of the current folder).

  • Rename a folder: Select the folder, click the name, then enter a new name. You can rename only folders you created.

  • Delete a folder: Select the folder, then choose Edit > Delete, or press the Delete key. You can delete only folders you created.

  • Move a folder: Drag the folder to another account. To make it a subfolder, drag it onto another folder. You can move only folders you created.

  • Move a note to another folder: Drag the note from the sidebar onto a folder.

  • Copy a note to another folder: Hold down the Option key while you drag the note.

If you create or edit notes while disconnected from the Internet, the updated notes are available to your accounts after you reconnect. If you don’t see your notes, make sure All Notes is selected in the folders list.

Your notes are stored in folders, corresponding to your accounts. If you don’t use Notes with any accounts, your notes are stored in the folder called On My Mac; those notes are available only on your Mac.

Any notes you previously created in Mail automatically appear in Notes.
