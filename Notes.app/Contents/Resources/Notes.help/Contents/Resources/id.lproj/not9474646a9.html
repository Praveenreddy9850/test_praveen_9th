Bantuan Catatan

Tuliskan hal-hal untuk diingat, atau simpan foto dan file dengan mudah. Jika Anda menggunakan akun Internet seperti iCloud atau Google dengan Catatan, catatan Anda akan tersedia di komputer atau perangkat lain di mana Anda menggunakan akun tersebut.

Sangat mudah memulai catatan pertama Anda: buka Catatan dan mulai mengetik. Untuk menulis catatan baru, klik tombol Tulis . Catatan menggunakan baris pertama catatan Anda sebagai judul.

Buka Catatan untuk saya

Menggunakan Catatan dengan akun Anda

  • Tambah akun untuk digunakan dengan Catatan: Pilih Catatan > Tambahkan Akun, pilih tipe akun seperti iCloud, lalu masukkan informasi mengenai akun Anda. Pastikan kotak centang Catatan dipilih untuk akun tersebut.

    Untuk menambahkan Catatan untuk suatu akun yang sudah digunakan di Mac Anda, pilih Catatan > Akun, pilih akun, lalu pilih kotak centang Catatan untuk akun tersebut.

  • Mematikan Catatan untuk suatu akun: Pilih Catatan > Akun, pilih akun, lalu batal pilih kotak centang Catatan. Catatan untuk akun tersebut tidak lagi ditampilkan di Mac Anda; untuk menampilkannya lagi, nyalakan Catatan untuk akun tersebut.

  • Mengubah akun default: Jika Anda menggunakan Catatan dengan lebih dari satu akun, Anda dapat memilih akun mana yang akan digunakan untuk catatan yang baru. Pilih Catatan > Akun Default.

Menuliskan catatan

  • Menambahkan dan menghapus gambar atau file lainnya: Seret gambar atau file teks ke catatan (Anda tidak dapat menambahkan file ke catatan di akun Exchange). Untuk menghapus gambar atau file, pilih gambar atau file, lalu tekan tombol Delete.

  • Mengubah font: Pilih Format > Font, lalu atur font default untuk semua catatan Anda, atau ubah font untuk teks yang dipilih di catatan. Dari menu Format, Anda juga dapat mengubah cara teks diratakan atau dilekukan.

  • Membuat daftar: Untuk memulai daftar berpoin dengan cepat, ketik * (asterisk), lalu tekan bar Spasi. Untuk membuat tipe daftar lain, pilih Format > Daftar.

  • Menemukan atau mengganti teks: Tekan Command-F. Untuk mengganti teks, pilih kotak centang Ganti, ketik teks pengganti, lalu ganti semua kemunculan teks asli, atau temukan dan ganti kemunculan teks khusus.

Jika menggunakan Handoff, Anda dapat beralih antara Mac dan perangkat lain saat menulis catatan. Untuk membuka catatan yang ingin dilanjutkan di Mac anda, klik ikon Catatan yang muncul di ujung Dock (di ujung kiri secara default). Untuk membuka catatan yang ingin dilanjutkan di perangkat iOS, gesek ke atas dari ikon aktivitas Catatan di layar terkunci perangkat, atau ketuk Catatan di layar multitugas.

Melihat catatan

  • Mengurutkan catatan: Pilih Lihat > Urutkan Menurut, lalu urutkan menurut tanggal atau judul.

  • Menghapus catatan: Pilih satu atau beberapa catatan, lalu pilih Edit > Hapus, atau tekan tombol Delete. Jika Semua Catatan dipilih di daftar folder, catatan dihapus dari folder masing-masing.

  • Menyimpan catatan sebagai dokumen PDF: Pilih catatan, lalu pilih File > Ekspor sebagai PDF.

  • Menyimpan catatan di desktop: Klik dua kali catatan di bar samping untuk membukanya di jendela yang terpisah, lalu pilih Jendela > Mengambang di Atas

  • Menemukan catatan: Gunakan bidang pencarian di bagian atas bar samping. Jika Anda memiliki lebih dari satu folder, Catatan akan mencari folder yang dipilih saja. Jika catatan yang diharapkan tidak ditemukan, klik Semua Catatan di daftar folder.

  • Menampilkan catatan di folder: Pilih folder di daftar folder. Untuk melihat semua catatan di suatu akun, pilih Semua folder untuk akun tersebut, seperti Semua iCloud. Untuk melihat semua catatan di semua akun, pilih Semua Catatan.

Menggunakan folder

  • Menampilkan atau menyembunyikan daftar folder: Pilih Lihat > Tampilkan Daftar Folder atau Lihat > Sembunyikan Daftar Folder.

  • Membuat folder: Gerakkan penunjuk melalui nama akun di daftar, lalu klik + yang muncul. Atau pilih File > Folder Baru (jika Semua Catatan dipilih, folder dibuat di akun default; jika tidak, folder akan dibuat di akun folder yang sedang dibuka).

  • Mengubah nama folder: Pilih folder, klik namanya, lalu masukkan nama baru. Anda hanya dapat mengubah nama folder yang sudah Anda buat.

  • Menghapus folder: Pilih folder, lalu pilih Edit > Hapus, atau tekan tombol Hapus. Anda hanya dapat menghapus folder yang sudah Anda buat.

  • Memindahkan folder: Seret folder ke akun lain. Untuk menjadikannya sebagai subfolder, seret folder ke folder lain. Anda hanya dapat memindahkan folder yang sudah Anda buat.

  • Memindahkan catatan ke folder lain: Seret catatan dari bar samping ke suatu folder.

  • Menyalin catatan ke folder lain: Tahan tombol Option saat Anda menyeret catatan.

Jika Anda membuat atau mengedit catatan saat terputus hubungan dari Internet, catatan yang diperbarui akan ada di akun Anda setelah Internet dihubungkan kembali. Jika Anda tidak melihat catatan Anda, pastikan Semua Catatan dipilih di daftar folder.

Catatan Anda disimpan di folder, sesuai dengan akun Anda. Jika Anda tidak menggunakan Catatan dengan akun apa pun, catatan Anda akan disimpan di folder yang berjudul Di Mac Saya; catatan tersebut hanya tersedia di Mac Anda.

Setiap catatan yang Anda buat sebelumnya di Mail otomatis akan muncul di Catatan.
