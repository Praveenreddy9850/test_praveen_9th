Справка Заметок

Запишите все, что надо не забыть, и держите нужные файлы и фото под рукой. Если для Заметок используются учетные записи типа iCloud или Google, то заметки будут доступны на всех компьютерах и других устройствах, где настроены такие учетные записи.

Начать создание первой заметки очень легко: просто откройте программу «Заметки» и начните вводить текст. Чтобы составить новую заметку, нажмите кнопку «Создать» . Первая строка заметки используется в качестве названия заметки.

Открыть Заметки

Использование программы «Заметки» с учетными записями

  • Как добавить учетную запись для использования с Заметками Нажмите «Заметки» > «Добавить учетную запись», выберите тип записи, например iCloud, затем введите данные своей учетной записи. Проверьте, что флажок «Заметки» установлен для данной учетной записи.

    Чтобы добавить Заметки в учетную запись, которая уже настроена на Вашем Mac, выберите «Заметки» > «Учетные записи», выберите нужную учетную запись, затем установите для нее флажок «Заметки».

  • Отключение Заметок для учетной записи. Выберите «Заметки» > «Учетные записи», выберите учетную запись и снимите флажок «Заметки». Заметки из отключенной учетной записи больше не будут отображаться на компьютере. Чтобы вновь увидеть их, включите Заметки для данной учетной записи снова.

  • Изменение учетной записи, используемой по умолчанию Если Заметки используются с несколькими учетными записями, можно выбрать, в какой учетной записи следует создавать новые заметки. Выберите «Заметки» > «Учетная запись по умолчанию».

Создание заметок

  • Добавление и удаление изображений и других файлов Потяните изображение или текстовый файл на заметку (нельзя добавлять файлы к заметкам в учетной записи Exchange). Чтобы удалить изображение или файл, выберите его и нажмите клавишу Delete.

  • Изменение шрифта. Выберите «Формат» > «Шрифт», затем задайте шрифт для использования по умолчанию во всех заметках или измените шрифт для выделенного текста в текущей заметке. В меню «Формат» можно также изменить отступы и выравнивание текста заметки.

  • Создание списков. Чтобы быстро начать маркированный список, введите * (звездочку) и нажмите клавишу пробела. Для создания списков других типов выберите «Формат» > «Списки».

  • Поиск и замена текста. Нажмите сочетание клавиш Command-F. Для замены текста установите флажок «Заменить»: введите текст замены и затем либо замените все экземпляры начального текста, либо найдите и замените конкретный экземпляр.

Используя Handoff, можно переключаться между компьютером Mac и другими устройствами при составлении заметок. Чтобы открыть заметку, переданную на Mac, нажмите значок Заметок, появившийся в правом конце панели Dock (в иных случаях он находится в левом конце). Чтобы открыть заметку, переданную на устройство с iOS, смахните вверх по значку активности Заметок на экране блокировки устройства или коснитесь значка Заметок на экране многозадачности.

Просмотр заметок

  • Сортировка заметок Выберите «Просмотр» > «Сортировать по», затем отсортируйте по дате или по названию.

  • Как удалять заметки Выделите одну или несколько заметок, затем выберите «Правка» > «Удалить» или нажмите клавишу Delete. Если при удалении в списке папок выбран вариант «Все заметки», то заметки удаляются из соответствующих папок.

  • Сохранение заметок как документов PDF Выберите заметку, затем выберите «Файл» > «Экспортировать как PDF».

  • Отображение заметок поверх всех окон Дважды нажмите заметку в боковом меню, чтобы открыть ее в отдельном окне, затем выберите «Окно» > «Поверх всех окон».

  • Поиск заметок Используйте поле поиска в верхней части боковой панели. Если папок с заметками несколько, поиск происходит только в выделенной в данный момент папке. Если нужной заметки нет в результатах поиска, нажмите «Все заметки» в списке папок.

  • Просмотр заметок в папке. Выберите нужную папку в списке. Чтобы отобразить все заметки для учетной записи, выберите папку «Все» для этой учетной записи, например «Все iCloud». Чтобы отобразить все заметки для всех учетных записей, выберите вариант «Все заметки».

Использование папок

  • Как показать или скрыть список папок. Выберите «Вид» > «Показать список папок» или «Вид» > «Скрыть список папок».

  • Создание новой папки Наведите курсор на название учетной записи в списке и нажмите +, который появится. Или выберите «Файл» > «Новая папка» (если выбран вариант «Все заметки», папка будет создана в учетной записи по умолчанию; в ином случае папка будет создана в учетной записи текущей папки).

  • Переименование папки. Выберите папку, нажмите ее имя и введите новое имя. Переименовывать можно только созданные Вами папки.

  • Удаление папки. Выберите папку, затем выберите «Правка» > «Удалить» или нажмите клавишу Delete. Удалять можно только созданные Вами папки.

  • Перемещение папки. Перетяните папку на другую учетную запись. Чтобы создать подпапку, перетягивайте на другую папку. Перемещать можно только созданные Вами папки.

  • Перемещение заметки в другую папку. Перетащите ее из бокового меню в нужную папку.

  • Копирование заметки в другую папку Перетягивайте заметку, удерживая клавишу Option.

Если заметки создаются или редактируются при отсутствии Интернет-соединения, то они станут доступны в учетных записях после подключения к Интернету. Если заметки не отображаются на экране, убедитесь, что в списке папок выбран вариант «Все заметки».

Ваши заметки хранятся в папках, соответствующих Вашим учетным записям. Если Заметки не используются с учетными записями, Ваши заметки хранятся в папке «На моем Mac», и они доступны только на компьютере Mac.

Все заметки, ранее созданные в программе Mail, автоматически появятся в программе «Заметки».
